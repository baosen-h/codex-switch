# Build Plan

## MVP

1. Provider management for Codex.
2. One-click provider activation.
3. Codex config generation and writing.
4. Session records linked to provider and workspace.
5. Resume helper for known session references.
6. Clean, fast desktop UI.

## Constraints

- Private repository.
- No license/open-source statement.
- No automatic backup workflow.
- Windows-first development.

## Key Risk

Codex session internals may not be fully controllable. The app must still record enough metadata to help locate and resume previous work even if full automatic session recovery is unavailable.
